DDALAB Application

This is the main application for DDALAB. To run it, you need Docker installed.

Quick Start with Docker

  1. Build the Docker Image: Open your terminal, navigate to the project root, and run: 
    docker build -t ddalab .
  2. Run with Docker Compose: From the same directory, execute: 
    docker compose up -d
  3. Access the Application: Open your web browser and go to http://localhost:3000
