# Visualization Guide

DDALAB provides interactive visualization for both raw data and analysis results.

## Time Series Viewer

### Basic Controls

- **Zoom**: Mouse wheel or pinch gesture
- **Pan**: Click and drag
- **Select**: Click on trace to select channel
- **Reset**: Double-click to reset view

### Channel Display

- Toggle channels via checkbox
- Reorder by drag-and-drop
- Color-code by type or custom

### Time Navigation

- Scroll bar for position
- Page forward/back buttons
- Jump to time input
- Keyboard: ←/→ for stepping

## Result Visualization

### Heatmap View

Visualize DDA results as a heatmap:

- X-axis: Delta values
- Y-axis: Channels
- Color: DDA magnitude

### Line Plot View

Individual channel results:

- Multiple channels overlaid
- Legend with channel names
- Tooltip for exact values

### Distribution View

Statistical distributions:

- Histogram of values
- Box plots per channel
- Comparative statistics

## Customization

### Color Schemes

Available palettes:

- Default (blue-red gradient)
- Viridis (perceptually uniform)
- Spectral (diverging)
- Custom (user-defined)

### Display Settings

Configure via Settings > Display:

- Line width
- Point markers
- Grid lines
- Background color

## Export

### Image Export

Export visualizations as:

- PNG (raster, web/print)
- SVG (vector, scalable)
- PDF (print quality)

### Options

- Resolution (DPI)
- Size (width x height)
- Include legend
- Include title
